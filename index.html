Git starter commands

Starter commands

git init
Initialise un repository local

git config --local user.name <user name>
Initialise npm de l'utilisateur

git config --local user.email <user email>
Initialise un email d'utilisateur

Add commands

git add <filename>
Ajoute un fichier dans l'index (il est tracké)

git add .
Ajoute les fichiers du dossier en cours dans l'index

git add -A
Ajoute TOUS les fichiers dans l'index (éviter)

Commit commands

git add index.html
git commit -m "commit de index.html
Ajoute puis commit le fichier index.html

git commit -m "your message"
Commit des fichiers de la zone staging vers le repository

git commit -a
Commit les fichiers de la zone staging vers le repository et ouvre l'éditeur par défaut (ici Vim) pour éditer le message

2 fois Ctrl+c puis :wq

git commit --help
Pour affichage de l'aide lié à "commit"

Redact commit message

  • Prendre le temps de rédiger son message
  • Ajouter un sujet
  • Limiter le message à 50 caratères
  • Ecrire sans accents
  • Répondre à la question "Pourquoi ?" plutôt que "Comment ?"

Affichage log

git log
Affiche les commits

git log -1
Affiche le dernier commit

git log --pretty=oneline --max-count=2
git log oneline Affiche sur une ligne les deux derniers commit

git log --pretty=oneline --since="5 minutes ago"
Affiche les commits effectués au cours des 5 dernières minutes

git log --oneline --until="5 minutes ago"
Affiche les commits effectués juqu'au 5 dernières minutes

git log --oneline --author="Nathalie W"
Affiche les commits effectués par un auteur

git log --oneline --pretty=format:"%h %cd %s (%an)" --since="1 days ago"
Affiche les commits %h pour le hash, %cd pour current date, %s pour afficher les commentaires, %an pour l'auteur...

Format de log

  • --pretty="..." format de sortie
  • Abbréviation de hash code commit
  • %d commit decoration
  • %ad commit date
  • %s commentaires
  • %an author name
  • --graph commit tree
  • --date=...

Version

git tag
Affiche des releases

git tag -a v0.0.1 "beta version"
Creation d'une release

Branche

git branch
Liste des branches

git branch<branchName>
Creation d'une branche

git checkout<branchName>
Connection à une branche / on switch sur une branche

git checkout -b <branchName>
Création et Connection à une branche

git checkout -d (ou -D) <branchName>
Suppression d'une branche - analyse des commit de la branche et -D force delete

git merge<branchName>
Fusion de branches à partir du noeud
