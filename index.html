Git starter commands

Starter commands

git init
Initialise un repository local

git config --local user.name <user name>
Initialise npm de l'utilisateur

git config --local user.email <user email>
Initialise un email d'utilisateur

Add commands

git add <filename>
Ajoute un fichier dans l'index (il est tracké)

git add .
Ajoute les fichiers du dossier en cours dans l'index

git add -A
Ajoute TOUS les fichiers dans l'index (éviter)

Commit commands

git commit -m "your message"
Commit des fichiers de la zone staging vers le repository

git commit -a
Commit les fichiers de la zone staging vers le repository et ouvre l'éditeur par défaut (ici Vim) pour éditer le message

2 fois Ctrl+c puis :wq

Redact commit message

  • Prendre le temps de rédiger son message
  • Ajouter un sujet
  • Limiter le message à 50 caratères
  • Ecrire sans accents
  • Répondre à la question "Pourquoi ?" plutôt que "Comment ?"
